## 🏗️ Build

### 🧱 Dependency

* git (to clone this repo)
* gcc
* make

### ❓ How to build

**Step 1:** Clone this repo

```sh
$ git clone https://github.com/K4zoku/2048.git
$ cd 2048
```

**Step 2:** Make

```sh
$ make
```

_If the build is successful, the binary file will now be located in `dist/bin`_
